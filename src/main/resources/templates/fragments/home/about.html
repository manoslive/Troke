Notre projet consiste en un site web transactionnelle. Dans ce site nous allons pouvoir faire la recherche de biens ou de services pour ensuite les échanger. Le but est vraiment de favoriser l'échange et non l'achat d'ou Trok-é pour troc électronique. Nous allons coder le tout en JAVA utilisant la philosophie MVC. L'IDE choisi est STS spring qui est une version modifier de éclipse et la base de données est MYSQL. De plus, nous aimerions avoir une version android de l'application web.
