Notre projet consiste en un site web de troc fait dans le cadre de la technique d'informatique de gestion au Cegep Lionel-Groulx. Dans ce site, l'usager peut remplir un inventaire d'objets ou de services qu'il pourra éventuellement échanger. Par la suite, l'usager peux faire la recherche de biens ou de services avec lesquels il peut débuter un troc. Lors de l'échange, l'interface permet un "Drag and Drop" facilitant la gestion des items. Après plusieurs confirmations entre deux usagers a l'aide d'un clavardage a l'interne, un courriel de confirmation est envoyé à chacun d'entre eux indiquant les items/services en troc. Le but est vraiment de favoriser l'échange et non l'achat, d'où Trok-é pour troc électronique. Cependant un ajustement monétaire lors d'un échange est possible. Nous avons coder le tout en JAVA SpringBoot / HTML / CSS / Javascript / Jquery UI / JPA / JPQL utilisant la philosophie MVC. L'IDE choisi est Intellij IDEA, et la base de données est MYSQL avec l'IDE WorkBench.
